# 🧮 Smart Calculator

A modern and interactive Calculator built with **HTML, CSS, and JavaScript**. This project was developed to strengthen my understanding of JavaScript logic, DOM manipulation, and creating responsive user interfaces.

## ✨ Project Overview

The Smart Calculator provides a simple and intuitive interface for performing mathematical calculations efficiently. Building this project helped me gain hands-on experience in implementing logical operations and improving frontend development skills.

## 🚀 Features

- ➕ Basic arithmetic operations
- 🖥️ Clean and user-friendly interface
- 📱 Responsive design
- ⚡ Fast and lightweight performance
- 💻 Cross-browser compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 What I Learned

While building this project, I improved my understanding of:

- JavaScript Logic Building
- DOM Manipulation
- Event Handling
- Responsive Web Design
- Frontend Project Structure
